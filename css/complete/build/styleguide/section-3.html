3.1 #permalink グリッド

横幅の合計を12カラムとして計算しています。 data-u-col属性にカラム数を指定し、分割する。
data-u-col属性はミディアムビュー、スモールビューにも引き継がれますがミディアムビュー、スモールビューでカラム数を変えたい場合はdata-u-col-md属性、data-u-col-sm属性で調整する。 c-gridはdata-grid-justify属性をcenterにする事で中央配置にすることが可能
c-grid__colはdata-grid-col-flex属性の有無でフレックスコンテナーにする事が可能。

Example
8 columns6 columns12 columns
4 columns6 columns12 columns
3 columns3 columns6 columns
3 columns3 columns6 columns
8 columns6 columns12 columns
4 columns6 columns12 columns
